The Individual Codes folder contains different files for encryption and decryption and a file with both feature combined. It is CLI based.

The Stegonography_with_UI.py file contains same feature with GUI.

The CLI based codes can be run and executed on any coding supporting platform.

Steps to download and run the steganography with UI code : 

	1. In terminal run commands : a. pip install opencv-python
				      b. pip install gradio
	
	2. Run the code after completing installation of above packages

	3. You will be redirected or given a local host link after running code. Go to the link on a browser.

	4. You are all set.
